🔑 Chiave API Groq

L'app usa Groq (gratuito, nessuna carta).

1. Vai su console.groq.com
2. Accedi con Google → "Create API key"
3. Copia la chiave (inizia con gsk_…) e incollala qui

📲 Installa su Android

Apri l'URL in Chrome su Android, poi tocca ⋮ → "Installa app" o "Aggiungi a schermata Home".

💾 Dati

Tutti i dati sono salvati sul tuo dispositivo. Non vengono mai inviati a nessun server.
