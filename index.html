🔑 Chiave API Groq

L'app usa Groq per generare definizioni ed esempi. È gratuito e non richiede carta di credito.

Per ottenere la tua chiave gratuita:
1. Vai su console.groq.com
2. Accedi con Google
3. Clicca "Create API key"
4. Copia la chiave (inizia con gsk_…) e incollala qui sotto

📲 Installa su Android

Puoi installare Parola come app sul tuo Android — senza passare dal Play Store. Basta seguire questi passi:

  • 1Scarica il file parole.html e caricalo su un server web gratuito come Netlify oppure aprilo tramite GitHub Pages. In alternativa usa il pulsante qui sotto se il browser lo supporta.
  • 2Apri l'URL dell'app in Chrome su Android.
  • 3Tocca i tre puntini ⋮ in alto a destra in Chrome.
  • 4Seleziona "Aggiungi a schermata Home" o "Installa app".
  • 5Conferma — l'icona di Parola apparirà nella tua schermata home come un'app vera.

💾 I tuoi dati

Tutte le parole, lo storico, i tag e le note sono salvati nel browser di questo dispositivo (localStorage). Non vengono mai inviati a nessun server.
